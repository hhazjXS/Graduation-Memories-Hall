### 需求描述

- 毕业留言回忆录

  一个允许用户创建数字版毕业纪念册的工具，用户可以上传照片、写留言、添加录音等，制作成一本完整的纪念册。

- 校园回忆地图
  一个基于地图的应用，用户可以在自己的学校地图上标记特定的地点，并附带一段回忆、照片或小故事。
